# Kampus-Naija
Kampus Naija is a multipurpose website built for students in the University. The website contain various section such as news, accommondation, ecommerce, election etc. It was built suing PHP and MYSQL. In the campus, various committee has been created to handle election. This committee can apply Kampus Naija Election section to help manage their electoral practise. The multipurpose website also include dashboard (admin panel) with which users can use to control the operation of the website.
